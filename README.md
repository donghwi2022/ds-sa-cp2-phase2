# 추천 시스템 모델링
## 1. 프로젝트 개요
  ### 1. 문제 인식
  다양한 미디어에서 추천 알고리즘을 활용하는중
  ### 2. 문제 정의
  다양한 추천 시스템을 활용하자
  ### 3. 가설 설정
  상품 정보를 바탕으로 추천을 진행하면 성능이 향상될 것이다.

  |데이터 전처리 및 <br> 시각화|2/1(수) ~ 2/2(목)|- 1차 데이터 전처리 진행 <br> - 다양한 시각화 진행||

## 3. 데이터셋 설명
  캐글에서 활용한 데이터셋 링크 + 컬럼별 내용, 최초 용량
## 4. 프로젝트 진행 내용
  전처리-시각화-전처리-모델링(기준모델과 구현모델 구분)-성능평가-모듈화
## 5. 프로젝트 결과
  기준 모델과 구현 모델 성능을 비교
